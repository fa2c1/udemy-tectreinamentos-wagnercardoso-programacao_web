Manual de Uso - Introdução das Aplicações utilizadas

No item acima, vamos realizar uma pesquisa onde as informações coletadas pela caixa de texto serão enviadas por um formulário (método POST) para uma página de tratamentos denominada receb.php.
Lá serão coletadas as variáveis coletadas pelo campo de pesquisa (use as palavras "office", "negocios" ou "marketing" para receber uma resposta para sua busca) que por meio de um Switch vai receber qual o dado coleto e vai relacionar a uma determinada resposta.

No item abaixo também haverá uma coleta de informações via formulário, porém neste momento em método POST para esconder as informações da URL.
Essas informações serão convertidas em variáveis que serão tratadas em uma estrutura de decisão onde caso a renda registrada seja maior ou igual a 2.000 o crédit será aprovado, caso não o crédito será recusado.
